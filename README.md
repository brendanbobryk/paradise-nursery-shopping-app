## Project Overview
Paradise Nursery Shopping Application allows users to explore a curated selection of houseplants, add them to a shopping cart, and view their total items and cost. The app emphasizes an intuitive shopping experience with categorized products, an easy-to-navigate header, and interactive cart management.

## Features
- **Landing Page**
  - Company name and description
  - Background image
  - "Get Started" button linking to the product listing page
- **Product Listing Page**
  - Six unique houseplants grouped into categories
  - Add to Cart functionality
  - Disabled button after adding to cart
  - Shopping cart icon updates with total items
- **Shopping Cart Page**
  - View all selected plants with thumbnail, name, price, and quantity
  - Increase or decrease item quantities
  - Delete items from the cart
  - Checkout button (displays "Coming Soon")
  - Continue Shopping button linking back to product listing
- **Header**
  - Displayed on all pages
  - Navigation links to Products and Cart
  - Shopping cart icon with total item count

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/brendanbobryk/paradise-nursery-shopping-app.git
   ```

2. Navigate to the project folder:
   ```bash
   cd paradise-nursery-shopping-app
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open the app
Visit http://localhost:5173 in your browser.
